\ Project Overview



The \AI Detective Escape Game\ is an interactive browser-based game where the player investigates a murder inside a mansion. The player interrogates suspects, collects clues, and uses AI-powered hints to deduce the killer before making the final accusation. The game simulates an AI detective assistant that evaluates clues and gives subtle suggestions, enhancing the immersive experience.







\ Game Features



\ Three main suspects: \Meera, Kabir, Raghav\.  

\ AI-generated hints based on clues and suspect behavior.  

\ Sequential interrogation and clue collection system.  

\ Decision-based gameplay: choices affect AI scoring but lead to a single final outcome.  

\ Final accusation phase with AI analysis summary.  

\ Simple, user-friendly interface built with \*Streamlit\*.  







\ How to Play



1\. Click \Start Investigation\ to begin.  

2\. Interrogate the suspects one by one.  

3\. Collect clues and pay attention to AI hints.  

4\. After exploring clues, decide when to make your accusation.  

5\. Confirm your final choice and see the AI analysis summary.  







\ AI Logic



\- Each suspect has a hidden \AI suspicion score\.  

\- Interrogations and clue selections increase suspicion scores.  

\- The AI evaluates contradictions between alibis and collected clues to suggest the most suspicious behavior.  

\- The AI summary helps the player understand which clues were most influential.  







\ Technologies Used



\- Python  

\- Streamlit for GUI and interactive web interface  

\- GitHub for code repository and version control  






\ Additional Features



\- Fully web-accessible; playable on laptops, tablets, and phones.  

\- Easy to extend with more suspects, clues, or alternate endings.  

\- Simple interface without images for fast loading and device compatibility.  






\ Conclusion



This project demonstrates a simple AI-based interactive game using Python and Streamlit. It combines decision-making, interactive storytelling, and AI hints, providing an engaging experience while showcasing the use of AI logic in game development.
